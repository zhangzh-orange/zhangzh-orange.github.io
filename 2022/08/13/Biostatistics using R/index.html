Chapter 4 Statistical inference

Statistical inference is a method of making decisions about the parameters of a population, based on random sampling.

4.1 Hypothesis testing

4 steps of hypothesis testing:

  1. State the hypotheses (null hypothesis H0 and alternative hypothesis HA).

  2. Deciding significance level and test method (often choose significance levels = 0.01 or 0.05).

  3. Calculating test statistic and P-value.

    The P-value is the probability of observing a sample statistic as extreme as the test statistic, assuming the null hypothesis is true.

  4. Whether to reject the null hypothesis.

4.2 Two-Tailed vs. One-Tailed Test

Two-tailed test:

  1. critical area of a distribution is two-sided
  2. whether a sample is greater than or less than a certain range of values

One-tailed test:

  1. critical area of a distribution is one-sided
  2. either greater than or less than a certain value, but not both

According to the testing requirements and professional knowledge, using one-tailed test as far as possible to improve the sensitivity of hypothesis testing.

4.3 Type I & Type II Errors

Type I error is a false positive conclusion；

Type II error is a false negative conclusion.

Example: Type I vs Type II error

You decide to get tested for COVID-19 based on mild symptoms. There are two errors that could potentially occur:

  • Type I error (false positive): the test result says you have coronavirus, but you actually don’t.
  • Type II error (false negative): the test result says you don’t have coronavirus, but you actually do.
